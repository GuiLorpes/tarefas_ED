1) O que é endereçamento direto?
    Um endereçamento direto cria um arranjo com poições para cada possivel chave
    de entrada, onde armazenamos nessas poições os valores da chave

2) O que é colisão?
    Quando duas chaves ao fazer uma mapeação delas geram a mesma posição

3) O que é uma função de dispersão? Qual é a principal caracteristica de uma 
   boa função de dispersão? 
    Uma função de dispersão é uma função que mapeia as chaves para suas 
    respectivas posições. Uma boa função hash é aquela que gera poucas colisões

4) O que são tabelas de dispersão?
    É uma estrutura de dados que usa uma função hash pra enviar os itens de uma 
    chave para a posição dada

6) Quais os cuidados devem ser tomados na implementação da remoção usando 
   endereçamento aberto?
    Ao remover do endereçamento aberto, caso consideramos o espaço que removemos
    como None, ele vai ignorar os espaços após ele, e isso pode fazer com que 
    ele não ache o elemento se estiver após.

7) Explique o conceito de fator de carga e porque ele é importante.
    O fator de carga é a média de itens por espaço do arranjo, ele é importante
    pois dependendo de quanto ele vale, o computador gasta menos memoria, e é
    rapido o suficiente para fazer operações em um bom tempo
